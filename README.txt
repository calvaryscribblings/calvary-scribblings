CALVARY SCRIBBLINGS - MULTI-PAGE SITE STRUCTURE
================================================

This is your complete multi-page website. Each page has its own URL!

FILE STRUCTURE:
===============
- index.html          (Homepage)
- news.html           (News & Updates page)
- inspiring.html      (Inspiring Stories page)
- creative.html       (Creative Writing landing page)
- flash.html          (Flash Fiction listing)
- short.html          (Short Stories listing)
- serial.html         (Serial Stories listing)
- stories/            (Folder for individual story pages)
  └── miss-lady.html  (Miss Lady story page)
- styles.css          (All the styling)
- scripts.js          (All the JavaScript)
- B4E36CD1-7C81-4ED0-BD27-63A125FDFD2D.png (Miss Lady cover - make sure this is uploaded!)

HOW TO UPLOAD TO GITHUB:
=========================
1. Delete ALL files currently in your GitHub repository (index.html, index2.html, README.md)
2. Upload ALL files from this folder to your repository root
3. Make sure the image B4E36CD1-7C81-4ED0-BD27-63A125FDFD2D.png is in the root (you uploaded this already)
4. Make sure the "stories" folder is uploaded with miss-lady.html inside it
5. Commit the changes

DIRECT LINKS WILL NOW WORK:
============================
- calvaryscribblings.co.uk/                    (Homepage)
- calvaryscribblings.co.uk/flash.html           (Flash Fiction page)
- calvaryscribblings.co.uk/stories/miss-lady.html (Direct link to Miss Lady!)

You can now share direct links to individual stories!
